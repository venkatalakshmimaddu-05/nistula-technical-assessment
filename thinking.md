Question B — The System Design

Beyond replying, the system should trigger a multi-step escalation workflow:

Classification: Detect this as a high-priority complaint (critical issue) using keywords like “no hot water” and “unacceptable.”
Alerting:
Notify the on-call maintenance team instantly (SMS/Call/WhatsApp).
Notify the property manager with high priority.
Ticket Creation:
Create a maintenance ticket tagged as urgent.
Log guest message, timestamp, and property ID.
Tracking & SLA:
Start a 30-minute response timer.
If no response:
Auto-escalate to senior manager
Trigger repeated alerts (call escalation)
Guest Updates:
Send periodic updates to reassure the guest.
Logging:
Log the issue, response time, resolution time, and outcome for analysis.
